# PollTech AI Voting System

An AI-based secure voting system with face authentication, role-based access, and real-time vote tracking.

## 🚀 Features

* Face authentication for voter verification
* Secure login system
* Role-based dashboards (Admin & Voter)
* Real-time vote tracking and analytics
* Clean UI built with React + Tailwind

## 🛠 Tech Stack

* React + TypeScript
* Vite
* Supabase  (Backend & Auth)
* Tailwind CSS

## 📦 Installation

```bash
npm install
npm run dev
```
